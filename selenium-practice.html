What is Selenium? +
Selenium is an open-source framework for automating web browsers. It supports multiple languages including Java, Python, C#, and JavaScript.
What is Page Object Model? +
POM is a design pattern that creates an object repository for web UI elements. It improves test maintenance and reduces code duplication.
Explicit vs Implicit Wait? +
Implicit wait applies globally for all elements. Explicit wait waits for a specific condition on a specific element using WebDriverWait.
